## 🔥 本周热门

**这些年，引入 React 方式的演变** — 作者研究了各种引入 React 的方式，并阐述了原由。令人振奋的消息是，未来将无需再通过 `import React from 'react'` 的方式引入 React。

Kent C. Dodds

**手把手带你创建一个 React 日志分析库** — 文章介绍了 Slack 中的三名工程师如何在他们的 React 桌面应用中进行日志分析。

Konstantin Savransky, Fabio Canache, Justin Huddleston

**Chakra UI 1.0：模块化 React 组件库** — Chakra UI 是一个支持无障碍访问（WAI-ARIA）和定制主题的 React 组件库。如今迎来了 v1 版本，该版本支持 React 17 以及 Emotion 11，并且支持主题自定义的 API。

chakra ui

**使用 react-konva 构建自己的便签（Stickers）组件** — 在你的日常工作中，可能便签无处不在。本文将教你如何使用 HTML5 Canvas 动画库 Konva 开发你的便签组件，并应用到你的项目中。其中，React Konva 是 Konva 的 React 版本。

Sarah Hamilton

**▶  跟 Shawn Wang 一起了解 React 的新特性** — Swyx 是 React 社区中备受瞩目的开发者。让我们来听听他和 TJ VanToll 的讨论，讨论内容涉及 React Core、React Routing 以及 React DevTools 的新特性。

React Round Up podcast

## 📘 教程与趣事

**如何使用 React Table 来构建表格** — 本文将从“零基础入门”的角度，教你如何上手 React Table，并附带详细且具体的示例。

Matthew Brown

**为什么我将不再使用 \`console.log()\` 来检查 React 状态更新** — 当需要进行调试时, `console.log()` 往往是开发者的第一选择，但其实你可以试试 `console.table()`，它是一个更棒的调试工具。

Aharon Hyman

**▶  TypeScript + React = ❤️** — 90 分钟让你爱上 TypeScript + React。本视频是 TSConf 2020 Workshop 中的一个主题，墙裂推荐。

Ben Ilegbodu

**用 GatsBy, React, Netlify 和 Amazon S3 构建播客网站** — `podcasthosting.org` 独家数据显示，目前（截止 2020 年 11 月份）全球已有 150 万个播客，超过 3400 万集。如果你也有自己的播客并且想提供个性化的服务，这篇文章可以提供一些帮助。

Vincent Tang

**在 React 应用中接入 Auth0 认证** — Auth0 是一个商业聚合授权服务，可以让用户使用他们更熟悉的第三方认证进行身份验证，比如 Google、 Facebook、Twitter。这篇文章阐述了如何在 React 应用中接入 Auth0 认证。

Nefe Emadamerho-Atori

**如何创建高效的 Contexts** — 作者从自己的角度出发谈及如何高效地使用 React Context。

Thomas Juster

## 🛠 代码和工具

**Downshift：用于构建可无障碍访问的 React 下拉框、组合框等组件的基础库** — 这个库的迭代非常频繁，以至于我们不经意间都错过了 6.0.0 版的发布。

PayPal et al.

**Rockpack：将搭建项目的时间从一周缩短至几分钟** — Rockpack 提供了服务端渲染、打包、校验、测试、日志记录以及本地化的功能，也许会成为 Create React App 的替代方案。

Sergey Aleksandrov

**Replay：一个受 React 启发而诞生的游戏引擎** — 我们有机会将 React 技能运用在游戏领域了！

Ed Bentley

**styled.macro：一个全新的 React Native 样式库** — 这是一个新出炉但是功能齐全的库。如果你希望你的 React Native 应用能够脱颖而出，那就抢在别人之前先试用下。

Ahmed T. Ali

**ui-neumorphism：“新拟物”设计风格的 React 组件库** — 我们很难去预测“新拟物”风格的未来，但是至少在当下我们很喜欢这样精致的外观，有一种罕见的感觉。

Ankit Kumar Pandit

**ESP8266 IoT Framework：由 React 提供 UI 界面的物联网框架** — 物联网是当下技术领域最火热的话题，但是 React 社区似乎无动于衷。本文提供了一个 React 与 ESP8266 技术结合的案例。

Maakbaas
