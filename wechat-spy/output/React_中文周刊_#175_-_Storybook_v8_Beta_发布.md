**版本发布：**

-   **react-intersection-observer v9.7** – 使用交叉观察者 API 来在元素进入或离开视口时采取行动。
    
-   **React Native Gesture Handler v2.15** – 暴露平台原生触摸 API，现在支持 macOS。
    
-   **react-jsonschema-form v5.17** – 使用 JSON Schema 声明性地构建表单。
    
-   **useHotkeys v4.5** – 在组件中使用键盘快捷键的 Hook。
    
-   **Vaul v0.9** – 无样式的抽屉组件，这是 **演示**。
    
-   **rc-drawer v7.1** – 另一个抽屉组件，以供对比。这是 **演示**。
    
-   **gridstack.js v10.1** – 快速构建响应式交互式仪表板。
    
-   **PrimeReact v10.5** – 大量的 UI 组件库。
    
-   **Facebook Relay v16.2**
