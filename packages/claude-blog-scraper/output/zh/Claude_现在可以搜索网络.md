# Claude 现在可以搜索网络

**_更新：_** _网络搜索现已向所有 Claude 计划全球开放。（2025 年 5 月 27 日）_

你现在可以使用 Claude 搜索互联网，以提供更新、更相关的回复。有了网络搜索，Claude 可以访问最新的事件和信息，从而提高其在受益于最新数据的任务上的准确性。

‍

当 Claude 将来自网络的信息整合到其回复中时，它会提供直接引用，以便你可以轻松核实来源。Claude 不会让你自己查找搜索结果，而是以对话格式处理并提供相关来源。这一增强功能通过实时见解扩展了 Claude 广泛的知识库，提供基于最新信息的答案。

使用带有网络搜索功能的 Claude 的流行方式：

*   **销售团队**可以通过分析行业趋势来了解关键举措和痛点，从而通过与潜在客户进行知情对话来改变客户规划并提高胜率。
*   **金融分析师**可以评估当前的市场数据、收益报告和行业趋势，以做出更好的投资决策并为金融模型假设提供信息。
*   **研究人员**可以通过在网络上搜索一手资料、发现新兴趋势并识别当前文献中的差距来构建更强有力的资助提案和文献综述。
*   **购物者**可以跨多个来源比较产品功能、价格和评论，以做出更明智的购买决定。

## 开始使用

网络搜索现已作为功能预览向美国的所有付费 Claude 用户提供。对我们的免费计划用户和更多国家/地区的支持即将推出。要开始使用，请在你的[个人资料设置](https://claude.ai/redirect/website.v1.511b92f9-8255-4545-abf4-9817c96cb7d2/settings/profile#feature-preview)中开启网络搜索，并开始与 Claude 3.7 Sonnet 进行对话。在适用时，Claude 将搜索网络以提供信息。
