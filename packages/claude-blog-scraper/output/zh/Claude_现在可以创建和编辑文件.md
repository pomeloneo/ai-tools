# Claude 现在可以创建和编辑文件

**更新：** 现已面向具有网络和出口控制的付费计划全面推出（2025 年 10 月 21 日）。

Claude 现在可以直接在 [Claude.ai](https://claude.ai/) 和桌面应用程序中创建和编辑 Excel 电子表格、文档、PowerPoint 幻灯片和 PDF。这改变了你与 Claude 协作的方式——不再只是接收文本回复或应用内工件（Artifacts），你可以描述你的需求，上传相关数据，并获得可直接使用的文件。

文件创建功能目前作为预览版向 Max、Team 和 Enterprise 计划用户提供。Pro 用户将在未来几周内获得访问权限。

### 你可以做什么

Claude 根据你的指令创建实际文件——无论是基于上传的数据工作、研究信息，还是从头开始构建。以下仅是几个示例：

*   **将数据转化为见解**：给 Claude 原始数据，获得经过清洗的数据、统计分析、图表以及解释关键点的书面见解的精美输出。
*   **构建电子表格**：描述你的需求——带有情景分析的财务模型、带有自动化仪表板的项目跟踪器，或带有差异计算的预算模板。Claude 会创建带有工作公式和多个工作表的电子表格。
*   **跨格式工作**：上传 PDF 报告并获得 PowerPoint 幻灯片。分享会议记录并获得格式化的文档。上传发票并获得带有计算的井井有条的电子表格。Claude 处理繁琐的工作并按照你需要的方式呈现信息。

无论你需要客户细分分析、销售预测还是预算跟踪，Claude 都能处理技术工作并生成你需要的文件。文件创建功能将通常需要编程专业知识、统计知识和数小时努力的项目转化为几分钟的对话。

### **它是如何工作的：Claude 的电脑**

在过去的一年里，我们看到 Claude 从回答问题转变为完成整个项目，现在我们正在让这种力量更容易获得。我们让 Claude 能够访问一个私有的电脑环境，在那里它可以编写代码并运行程序来生成你需要的文件和分析。

这将 Claude 从顾问转变为积极的合作者。你带来背景和策略；Claude 在幕后处理技术实现。这展示了我们前进的方向：通过对话让复杂的多步骤工作变得触手可及。随着这些能力的扩展，想法和执行之间的差距将继续缩小。

### **开始使用**

要开始创建文件：

1.  在 [Settings > Features > Experimental](https://claude.ai/settings/features)（设置 > 功能 > 实验性）下启用“Upgraded file creation and analysis”（升级的文件创建和分析）
2.  上传相关文件或描述你的需求
3.  通过聊天引导 Claude 完成工作
4.  下载完成的文件或直接保存到 Google Drive

![开启开关的图像](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/68e95d910d03dc12cdc4df7c_1cbf5333ab0784b215a2450fb8dd95cd2e207aea-3840x2160.png)

先从简单的任务开始，如数据清洗或简单的报告，一旦你熟悉了 Claude 如何处理文件，再尝试复杂的项目，如财务模型。

此功能允许 Claude 访问互联网以创建和分析文件，这可能会使你的数据面临风险。使用此功能时请密切监控聊天。[了解更多](https://support.anthropic.com/en/articles/12111783-create-and-edit-files-with-claude)。
