# 为 Claude 构建 Skills 的完整指南

自 [10 月份推出 Skills](https://claude.com/blog/skills) 以来，我们看到了来自开发者的浓厚兴趣，他们希望 Claude 能够一致地遵循特定的工作流程；还有那些正在自动化重复性任务（如文档创建或研究流程）的高级用户；希望标准化 Claude 在整个组织中运作方式的团队；以及正在将其集成与可靠工作流配对的 MCP 连接器构建者。

共同点是：人们希望获得关于构建有效 Skills 的更详细指导——无论是用于独立工作流还是 MCP 增强的集成——从规划和结构到测试和分发。

我们编写了本指南来填补这一空白。它涵盖了从技术要求和 Skill 结构到测试方法和常见故障排除模式的所有内容。无论你是构建与 Claude 内置功能配合使用的独立 Skills，还是增强 MCP 集成，你都会发现这些指导非常相关——对于那些有需要的人，还有一个关于 MCP + Skills 的专门部分。

**你将学到什么：**

*   Skill 结构的技术要求和最佳实践
*   独立 Skills 和 MCP 增强工作流的模式
*   在不同用例中行之有效的模式
*   如何测试、迭代和分发你的 Skills

如果你知道你想自动化的前 2-3 个工作流，预计使用 skill-creator 构建和测试你的第一个工作 Skill 大约需要 15-30 分钟。

**本指南适用于：**

*   希望 Claude 一致遵循特定工作流程的开发者
*   在集成中添加工作流知识的 MCP 连接器构建者
*   自动化重复性任务的高级用户
*   希望标准化 Claude 在整个组织中运作方式的团队

点击 [这里](https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf?hsLang=en) 下载指南。
