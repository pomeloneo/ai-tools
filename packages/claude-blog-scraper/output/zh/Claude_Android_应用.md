# Claude Android 应用

新的 Claude Android 应用程序将 Claude 的力量——包括我们最强大的模型 Claude 3.5 Sonnet——带给了 Android 用户。该应用程序是免费的，并且可以通过所有计划访问，包括 Pro 和 Team。

Claude Android 应用程序的工作方式与 Claude 在 iOS 和网络上的工作方式完全相同，这意味着你可以获得：

*   **多平台支持**：在 [Web](https://claude.ai)、[iOS](https://apps.apple.com/us/app/claude-by-anthropic/id6473753684) 和 [Android](https://play.google.com/store/apps/details?id=com.anthropic.claude) 应用程序之间继续与 Claude 的对话
*   **视觉能力**：拍摄新照片或上传文件以进行实时图像分析
*   **多语言处理**：实时语言翻译，帮助沟通或翻译你周围世界的各个方面
*   **高级推理**：Claude 可以帮助你解决复杂问题，例如在旅行时分析合同或进行市场研究以准备会议

## 随时随地与 Claude 交谈

将 Claude 用于工作或娱乐。无论你是在会议间隙起草商业提案、在旅行时翻译菜单、在购物时集思广益挑选礼物，还是在等待航班时撰写演讲稿，Claude 随时准备为你提供帮助。

## 开始使用

要开始使用 Claude Android 应用程序，请在 [Google Play](https://play.google.com/store/apps/details?id=com.anthropic.claude) 上下载它。
