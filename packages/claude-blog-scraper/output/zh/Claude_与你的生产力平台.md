# Claude 与你的生产力平台

Claude 现在与 Microsoft 365 集成，并提供跨连接工具的企业级搜索。

Claude 与你每天使用的生产力工具协同工作，将文档、电子邮件和日历中的上下文直接带入你的对话中。花更少的时间收集信息，将更多的时间用于推动影响力的决策和工作。

## Claude 的 Microsoft 365 连接器

使用我们的 MCP 连接器将 Claude 与 Microsoft 365 连接。通过访问你的文档、通信和日历，Claude 获得了更有效协作的上下文——帮助你推理复杂问题，做出明智的决定，并更快地采取行动。

Claude 支持：

*   **Microsoft SharePoint 和 OneDrive**：跨站点和库搜索和分析文档，从项目规范到战略计划，无需手动将文件上传到 Claude。
*   **Microsoft Outlook**：访问电子邮件线程，分析通信模式，并从你的通信中提取见解，以了解项目状态、客户反馈或团队一致性。
*   **Microsoft Teams**：搜索聊天对话、频道讨论和会议摘要，以浮现决策、跟踪项目更新并了解团队正在讨论的内容。

## 企业级搜索 (Enterprise search)

![显示各种连接的应用程序图标的工作流图表](https://cdn.prod.website-files.com/68a44d4040f98a4adf2207b6/690269bad95a265f46e2268b_image.png)

我们还通过一个专用的、以你公司名称命名的共享项目，使访问公司知识变得更容易。该项目包括自定义提示，帮助 Claude 在你公司的工具中有效地进行搜索，以及连接重要工作应用程序的简单设置步骤。一旦管理员设置好并连接了必要的工具，组织中的每个人都会自动获得访问该项目的权限。

当你问 Claude 一个问题时，它可以在一个地方搜索你公司连接的数据源，这样你得到的答案不仅来自你自己的文件，还来自你组织的集体知识。例如，向 Claude 询问你公司的远程工作政策，并在一份详细的报告中获得来自 SharePoint 中的人力资源文档、Outlook 中的电子邮件讨论以及各种来源的团队指南的信息。

企业级搜索对于新团队成员入职、回答战略问题（如分析客户反馈中的模式）以及快速确定就任何主题咨询哪位内部专家特别有价值。

## 开始使用

Microsoft 365 连接器今天可供所有 Claude Team 和 Enterprise 计划客户使用。管理员必须先为其组织启用连接器，然后个人用户才能使用其帐户进行身份验证。

企业级搜索已为所有 Team 和 Enterprise 组织启用。管理员需要先自定义项目并为其组织策划数据源，团队才能使用它。
