## 🔧 代码和工具

**Draft.js：Facebook 的富文本编辑框架** — 距上次推荐 Draft.js 这个富文本编辑框架已经有年头了（准确地说应该是在 2016 年末）。经过一系列更新后，我们认为是时候重新了解它了。开始使用的最好方式就是试用它。

Facebook

**react-use-pip：画中画功能的 React Hooks** — 有人可能把 PIP（画中画）作为新选购电视时的必需条件，这样你可以一次观看两场周六赛事。如果是这样，你就可以和自己约会。老树新花，PIP 现在是指在其他内容上浮动播放视频的能力。它也被一些现代浏览器所支持。

Daw-Chih Liou

**react-scroll-sync：同步你的滚动元素** — 一个悲伤的故事是大家阅读文档，很少会顺序阅读。我们认为这个组件开启了有趣的方式，以一种即保持你想要的顺序，同时又迎合受众阅读内容顺序的方式展示信息。此示例清晰地描绘了这个想法。

Andrey Okonetchnikov

**react-awesome-query-builder：为你的查询构建添加规则** — 大部分人在想搜索内容时，会在地址栏简单输入随机的词汇以开始搜索，但如果用户在某些场景下能用一个结构化的方式创建查询是非常有帮助的 — 而这个组件就能帮你做到这些。

Denis Oblogin

**Chonky：React 中实现文件浏览器的映射** — 20 世纪 70 年代初期 Alan Kay 在 Xerox PARC 创造了在桌面上浏览文件和目录的概念。今天，可以说每一个技术用户都精通它。那么如何使用 React 组件来实现它呢？

Tim Kuzhagaliyev
