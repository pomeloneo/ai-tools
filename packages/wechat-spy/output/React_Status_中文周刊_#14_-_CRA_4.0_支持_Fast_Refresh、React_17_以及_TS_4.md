## 🔥 本周热门

**Create React App 发布了 4.0 版本** — 距离 Create React App（快速创建新的 React 应用的工具）上一次大版本发布已经过去一年多了，好在这个版本带来了很多新特性，等待就是值得的。

4.0 带来了很多新特性，支持了 Fast Refresh（hot reload 的替换方案）、React 17 以及 TypeScript 4。同时，也支持了 ESLint 7、Jest 26 和 Web Vitals 并对 PWA/Workbox 进行了改进。

Facebook React Team

**Next.js 10 发布：图片优化，路由支持国际化，支持 React 17** — Next.js 的下一个大版本也同样值得期待。它将会支持 Analytics（集成了 Web Vitals），并与 BigCommerce 合作实现了 Next.js Commerce。

Vercel

**用 Reactime 6.0 实现状态的时间旅行** — 许多调试状态管理的问题不仅需要关心当下的状态是什么，更重要的是，还需要关心过去的状态是什么，是什么导致了现在的状态。让我们和 Reactime 一起回到过去，一探究竟。

Vincent Nguyen
